# Images Directory

## Crown Logo

**File needed:** `crown-logo.png`

**Instructions:**
1. Save the golden crown image as `crown-logo.png`
2. Place it in this directory: `assets/images/crown-logo.png`
3. Recommended size: 200x200 pixels (will auto-resize to 48x48)
4. Format: PNG with transparent background
5. Quality: High resolution for retina displays

**Current Status:**
- Logo image slot ready in homepage
- Using IMG tag for better quality
- Same logo will be used in:
  - Header (all pages)
  - Footer (all pages)
  - Admin panel
  - Favicon

**Alternative:**
If you don't have the PNG file:
- You can use the crown image you shared
- Save it as crown-logo.png
- Upload to this folder
- Website will automatically show it

**Note:** Until the image is added, text logo "Crown" will display as fallback.
